# Portfolio Website Project Plan: Pongpak A - Data Analyst

## 1. Objective
Create a modern, dark-themed, data analyst portfolio website hosted on GitHub Pages, featuring professional branding, three project showcases, and clear contact information.

## 2. Technical Stack
- **Structure:** Static HTML5
- **Styling:** CSS3 (Custom Dark Theme, JetBrains Mono font)
- **Interactivity:** Vanilla JavaScript
- **Hosting:** GitHub Pages

## 3. Project Structure
- `index.html`: Main landing page structure.
- `style.css`: Theme configuration, layout, and typography (JetBrains Mono).
- `script.js`: Minor interactivity (e.g., smooth scrolling, project filter).
- `/assets`: Folder for project images and PDFs.

## 4. Implementation Steps
### Phase 1: Setup
1. Create a GitHub repository named `pongpak-a.github.io`.
2. Initialize with a README and clone locally.
3. Set up the file structure (`index.html`, `style.css`, `script.js`, `/assets`).

### Phase 2: Core Development
1. **Hero Section**: Implement name "Pongpak A" (Data Analyst) with modern typography.
2. **Project Section**: Create a grid layout to showcase three data analysis projects.
3. **Contact Section**: Add professional links (Email, LinkedIn, GitHub).
4. **Styling**: Apply the dark theme and JetBrains Mono typography.

### Phase 3: Deployment
1. Commit all files to the main branch.
2. Push to GitHub.
3. GitHub Pages will automatically serve the site from the root directory.

## 5. Verification
- Validate the site renders correctly in Chrome/Firefox.
- Ensure all links (GitHub, LinkedIn) are active.
- Confirm "dark theme" and "JetBrains Mono" fonts are applied correctly.
